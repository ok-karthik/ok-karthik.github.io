# Karthik Orugonda - Portfolio

Welcome to the source code for my professional portfolio. 

I am a Senior Platform Engineer & SRE with 10+ years of experience building cloud-native platforms, internal developer tooling, and resilient infrastructure across AWS, Azure, and GCP. 

This portfolio is designed to showcase my technical expertise, career history, and featured projects as I transition into AI Infrastructure and AI Platform Architect roles.

## Technology Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **UI Components**: Radix UI / shadcn
- **Theming**: next-themes (Dark/Light mode support)

## Local Development

**Prerequisites**: Node.js 22+ and pnpm.

To run this project locally:

1. Install dependencies:
```bash
pnpm install
```

2. Start the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js App Router core (layout, global styles, theme provider).
- `components/` - React components including the interactive DevOps Terminal, Neural Mesh background, and animated sections.
- `PROJECT_CONTEXT.md` - Context document detailing architectural and design choices.
