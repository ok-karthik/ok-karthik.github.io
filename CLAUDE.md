# Agent Instructions
Please read the modular rule files located in the `.agent/rules/` directory and the skills in `.agent/skills/` before modifying this codebase.
They contain the strict UI, architectural, and testing constraints for this project.

## Core Behaviors
1. **Always `git pull`**: Always perform a `git pull` before making any changes to ensure you are working with the latest codebase.
2. **Explain Technical Details**: When making changes, fixes, or implementing new features, always break down the technical details and explain them clearly to the user. Do not just blindly write code—educate the user on the "why" and "how".
